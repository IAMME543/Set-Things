# Open But Polished

1. Client side decorations
    I have chosen client side decorations when starting out this project because it allows the space that would have been taken up by a (admitidly not nice looking) SSD from sway to be used more effectively with menus. However this also not introduces the ability for top bars to be inconsistent, which means that implementations for a CSD should be defined and standardised.

2. Lucide icon library
    given the scope of this project i see no reason to create my own common icon library. i have chosen lucid because a random person on reddit said so and i dont entirely know how much it matters at this stage.
